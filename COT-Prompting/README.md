An educational single-page website that teaches the **Chain-of-Thought (CoT) prompting** pattern using everyday cooking examples. Built as part of a Prompt Engineering & AI course assignment.

## Overview

This project explains how Chain-of-Thought prompting works by treating the AI like a **thoughtful home cook** that plans a recipe step by step instead of jumping straight to the final dish. It covers:

- What Chain-of-Thought prompting is  
- Why it matters in NLP and LLMs  
- How it improves reasoning and user experience  
- A cooking assistant scenario with practical prompts and responses  
- Quiz and exercise sections for self-check

## Features

- Clear, student-friendly explanation of CoT using a cooking theme  
- Sections for:
  - Definition and core concepts  
  - Purpose and importance in NLP  
  - Real-world cooking assistant example  
  - Example prompts and simulated AI responses  
  - Analysis of strengths/weaknesses  
  - Quiz and hands-on exercise  
- Pure **HTML/CSS/JS** (no backend or frameworks)  
- Works locally in any modern browser
